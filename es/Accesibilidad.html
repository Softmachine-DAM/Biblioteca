Compromiso con la Accesibilidad

Desde la biblioteca Municipal de Muskiz, nos comprometemos a garantizar que nuestro sitio web sea accesible para todos los usuarios, independintemente de sus capacidades. Estamos trabajando activamente para cumplir con las pautas de accesibilidad web (WCAG) y ofrecer una experiencia incluisva para todas las personas, incluidas aquellas con discapacidades.

Cumplientos de las Pautas de Accesibilidad

Nuestro sitio web esat diseñado para cumplir con las pautas de accesibilidad para el contenido web (WCAG), nivel AA, para garantizar que el contenido sea accesible para la mayoria de los usuarios, incluidos aquellos con discapacidades visuales, auditivas, motoras o cognitivas. Esto incluye:

  • Contraste de Color: El textotiene un alto contraste con el fondo para garantizar que sea legible, incluso para personas con baja vision o daltonismo
  • Textos Alternativos: Todas las imagenes en el sitio tienen descripciones detalladas (atributo "alt") para usuarios de lectores de pantalla, proporcionando contexto y descripcion para cada imagen

Compatibilidad con Tecnologias de Asistencia

Nuestro sitio web es completamente compatible con diversas tecnologías de asistencia, tales como:

  • Lectores de Pantalla: El contenido de nuestro sitio está estructurado de manera que sea fácilmente comprendido por programas como JAWS, NVDA y VoiceOver. Los encabezados, enlaces y formularios están claramente etiquetados para facilitar la navegación
  • Navegación por Teclado: Todo el sitio web es accesible utilizando solo el teclado. Los usuarios pueden navegar por todas las secciones y formularios utilizando la tecla Tab y otros atajos sin la necesidad de un ratón

Contenido Multimedia Accesible

Nos aseguramos de que todos los recursos multimedia sean accesibles para todos los usuarios:

  • Subtítulos: Todos los videos en nuestro sitio web incluyen subtítulos para usuarios con discapacidades auditivas.
  • Transcripciones: Proporcionamos transcripciones completas para todo el contenido de audio y video disponible en el sitio.

Opciones de Accesibilidad en la Página

Para garantizar una mejor experiencia de navegación, hemos implementado las siguientes opciones personalizables:

  • Modo de Contraste Alto: Si necesitas mejorar la visibilidad del contenido, puedes activar un modo de contraste alto haciendo clic en el ícono de accesibilidad en la parte superior del sitio.
  • Ajuste de Tamaño de Texto: Ofrecemos una opción para aumentar o disminuir el tamaño del texto en todo el sitio, permitiendo a los usuarios ajustar la legibilidad según sus necesidades.

Navegación Clara y Consistente

Hemos diseñado nuestra página con una estructura lógica y jerárquica:

  • Encabezados Claros: Usamos una jerarquía adecuada de encabezados (h1, h2, etc.) para que los usuarios de lectores de pantalla puedan navegar fácilmente por el contenido. haciendo clic en el ícono de accesibilidad en la parte superior del sitio.
  • Enlaces Descriptivos: Todos los enlaces tienen descripciones claras y específicas para que los usuarios comprendan su propósito sin tener que hacer clic.
